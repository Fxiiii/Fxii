Cent Browser | 绝对最强主力浏览器

发表于 2020-04-01 更新于 2020-11-22 分类于 必备软件

百分浏览器 Cent Browser

吖燊推荐

  • Chrome有的它都有，Chrome没有的它还有！！
  • 简约！！干净！！杜绝一切花里胡哨！！

官网介绍

  • 极速内核：基于最新的Chromium开源内核进行优化开发，充分继承其高速安全的特性。——小白们不用管这个，你就知道它跟Google chrome一样就好
  • 鼠标手势：无所不能的鼠标手势让您的操作更加高效——说实话，这个用到的真不多，主要是需要按住“右键”来操作，看个人习惯吧！
  • 超级拖拽：轻轻一拖，就可以实现复制搜索下载等高级功能——这个是真方便！！
  • 内存优化：延迟加载后台标签+强制释放内核缓存，可以大幅度减少内存占用——小白们依然不用管这个
  • 小号标签：想登陆多个账号不串号，Ctrl+M来试试——其他浏览器都需要外开无痕，Cent Browser同一窗口下小号标签想开多少开多少！！
  • 高度定制：大量实用的可定制特性——牛逼克拉斯的地方！！

个人体会

  • 超级拖拽是真方便！！几乎是用到最多的功能，浏览网页时候，这个图片不错，根本不用什么右击→另存为，鼠标按住直接向下一拖自动下载。网页有链接，鼠标按住直接向右一拖后台打开，既不影响当前网页浏览，又能在下一标签页自动打开新链接。
  • 双击关闭标签页，标签上没有×，没有任何多余按钮，杜绝一切误触！！
  • Google账户服务，完美支持，根本不存在同步不了的问题。
  • Chrome所有插件，完美支持。
  • 更多优雅细节难以言表！！
